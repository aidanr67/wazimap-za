Using {{ WAZIMAP.name }}

Finding places

Type in the name or address of a place. Then select the location you are interested in from the dropdown. Alternatively, click on the map area you want to go to. The + and - buttons zoom in an out.

Exploring data

Statistics are broken down into categories, e.g. Service Delivery. Find a key figure for a sub-category on the left. Hover over graphs to see comparisons with the province and country averages.

About {{ WAZIMAP.name }}

{{ WAZIMAP.name }}, from the Xhosa word ulwazi for knowledge, is a joint project by Ilifa Labantwana, Media Monitoring Africa and Code for South Africa that provides easy access to data relating to Early Childhood Development.

Shoulders of Giants

{{ WAZIMAP.name }} is based on the amazing Census Reporter and is completely open source. You can fork Wazimap on Github and build one for your country.
